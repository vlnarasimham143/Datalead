# Data Lead Portfolio

A modern, responsive portfolio for a data delivery leader with 19 years of leadership experience. Built with Next.js and React.

## Personalise it

Open `app/page.tsx` and replace:

- `YOUR NAME` with your name
- `hello@yourdomain.com` with your email
- The LinkedIn URL with your profile URL

Also update the page title in `app/layout.tsx`.

## Run locally

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Deploy

Push this folder to GitHub, then import the repository into Vercel. Vercel will detect Next.js and deploy it with the default settings. Netlify also supports this project directly.
